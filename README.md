# tool-belt
Uma coleção de ferramentas de produtividade reunidas em um só lugar! Experimente e use as que mais fazem sentido para você.

## Pomodoro

O pomodoro é conhecido por ser uma ferramenta de gestão de tempo que funciona em blocos de tempo. Originalmente, eram definidos blocos de 25 minutos de foco para 5 minutos de descanso. Altere como quiser!

### Funcionalidades
- Start
- Stop
- Reset
